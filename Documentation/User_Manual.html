📖Introduction & Overview

Welcome to SeedBetArt Ai Bot, a powerful WordPress plugin designed to provide professional-grade betting analysis and AI-powered recommendations for football matches.

✨Key Features

  • Value Bets: Identifies matches with positive expected value (EV)
  • Lay The Draw (LTD): Suggests matches where laying the draw is statistically advantageous
  • Under 2.5 Goals: Predicts low-scoring matches with odds analysis
  • AI Analysis: OpenAI-powered detailed match analysis
  • Multi-Theme Support: Professional theme system with custom designs
  • Real-Time Updates: Hourly automated odds and match data refreshes
  • Responsive Design: Works perfectly on desktop, tablet, and mobile

This manual will guide you through every step of installation, configuration, and usage.

🔑API Setup & Validation

SeedBetArt Ai Bot requires three external APIs to function properly. This section guides you through obtaining and validating each API key.

1. Odds-API.io (Required)

Purpose

Provides current betting odds and match data from multiple sportsbooks

Setup Steps

  1. Visit https://www.odds-api.io/
  2. Click "Sign Up" and create a free account
  3. Go to your dashboard and find your API Key
  4. Copy the API key (it starts with a long alphanumeric string)

Pricing

  • Free: 100 requests/month (limited testing)
  • Paid plans available for production use

2. API-Football.com (Required)

Purpose

Provides team information, match statistics, and team badges

Setup Steps

  1. Visit https://www.api-football.com/
  2. Sign up for a free account
  3. Navigate to your dashboard
  4. Copy your API Key

Pricing

  • Free: 100 requests/day
  • Sufficient for most users

3. OpenAI API (Optional)

Purpose

Enables AI-powered match analysis and betting recommendations

Setup Steps

  1. Visit https://platform.openai.com/
  2. Create an OpenAI account or sign in
  3. Go to API Keys section
  4. Click "Create new secret key"
  5. Copy and save the key securely

Pricing

  • Usage-based pricing
  • Recommended: Start with GPT-5.4 Mini for the best balance of quality and cost

Validating API Keys in WordPress

1 Navigate to Settings

In WordPress Admin: SeedBetArt Ai Bot → ⚙️ Settings

2 Enter API Keys

Paste each API key into its corresponding field (Odds-API, Football-API, OpenAI API)

3 Click Validate Buttons

Click the ✓ Validate button next to each API key

4 Check Results
  • Green ✓ button: API key is valid and working
  • Red ✗ button: Shows specific error message explaining the issue

Common Validation Errors

Error Message Solution
"Invalid API key" Double-check the key copied from the provider. Ensure no extra spaces.
"Rate limit exceeded" Upgrade your API plan or wait for the limit to reset.
"Connection timeout" Check your internet connection. Verify your server has HTTPS enabled.
"Unauthorized" The API key may have been revoked. Generate a new one from the provider.
Do not share your API keys. Treat them like passwords. Anyone with your keys can use your API quotas and incur costs.

📝Using Shortcodes

Shortcodes allow you to display betting recommendations on any WordPress page or post. Add these shortcodes directly in the page/post editor.

Available Shortcodes

1. Value Bets

[bsp_v2_value_bets]

Purpose: Displays matches with positive expected value (EV)

Shows: Match details, odds, EV percentage, confidence score

Best For: Highlighting profitable betting opportunities

2. Lay The Draw (LTD)

[bsp_v2_ltd]

Purpose: Displays matches where laying the draw is advantageous

Shows: Match, draw probability, recommended odds

Best For: Advanced betting strategies

3. Under 2.5 Goals

[bsp_v2_under25]

Purpose: Displays low-scoring match predictions

Shows: Match details, expected goals, confidence

Best For: Conservative betting approach

4. AI Analysis

[bsp_v2_ai_analysis sport="football" league="Premier League"]

Purpose: Displays AI-powered match analysis

Requires: OpenAI API key configured

Shows: Detailed AI-generated match analysis and recommendations

How to Add Shortcodes to Your Site

1 Create or Edit Page

Go to WordPress Pages or Posts and create a new page or edit existing one

2 Open Block Editor

Click in the editor area where you want the recommendations to appear

3 Add Shortcode Block

Add a "Shortcode" block (or paste the shortcode directly in the editor)

4 Paste Shortcode

Copy one of the shortcodes above and paste it into the block

5 Publish

Save and publish the page. The recommendations will display automatically.

Shortcodes automatically validate that all APIs are configured before displaying. If an API is not validated, users will see a helpful error message.

🔧Troubleshooting

Shortcodes Not Displaying Data

Problem

Shortcodes appear but show an error message like "APIs not validated"

Solution

  1. Go to SeedBetArt Ai Bot → ⚙️ Settings
  2. Verify all API keys are entered correctly
  3. Click the ✓ Validate button for each API
  4. Ensure all three APIs show green validation status
  5. Check Activity Log for specific error messages

Widgets Not Showing

Problem

Widget displays as empty or with loading spinner that never completes

Solution

  1. Clear your browser cache (Ctrl+F5 or Cmd+Shift+R)
  2. Check browser console for JavaScript errors (F12)
  3. Verify all APIs are validated in Settings
  4. Try disabling other plugins temporarily to check for conflicts
  5. View Activity Log for error details

Theme Not Applying

Problem

Selected theme doesn't appear after clicking "Apply Theme"

Solution

  1. Clear browser cache (Ctrl+F5)
  2. Try applying theme again
  3. Check if browser allows JavaScript (console should show [BSP V2 Admin] logs)
  4. Try different theme
  5. Check Activity Log for theme switch events

API Validation Errors

Problem

API validation returns specific error messages

Common Errors & Solutions

Error Fix
"Invalid API key" Copy key again carefully, check for extra spaces
"Rate limit exceeded" Upgrade API plan or wait for daily reset
"Network error" Check internet, verify HTTPS is enabled on server
"Service unavailable" API provider may be down. Try again later.

Slow Performance

Problem

Dashboard or shortcodes load slowly

Solution

  1. Check API response times (visible in Activity Log)
  2. Increase cache TTLs in plugin settings
  3. Reduce number of items displayed in widgets/shortcodes
  4. Check WordPress admin performance (Ctrl+Shift+K console)
  5. Consider using a caching plugin for WordPress

Accessing Activity Log

For detailed error information:

  1. Go to SeedBetArt Ai Bot → 📋 Activity Log
  2. View all system events and error messages
  3. Or check log files in plugin's `/logs/` directory

❓Frequently Asked Questions

Q: Do I need all three APIs to use the plugin?

A: Odds-API and Football-API are required. OpenAI is optional but needed for AI analysis features. Without OpenAI, you still get value bets, LTD, and Under 2.5 predictions.

Q: How often is data updated?

A: Odds are updated hourly via automated cron jobs. Match data updates similarly. You can manually refresh from the dashboard.

Q: Can I use multiple themes?

A: Yes, but only one theme is active at a time. You can switch themes anytime from SeedBetArt Ai Bot → 🎨 Themes.

Q: What are "Value Bets"?

A: Value bets are matches where the odds offered are higher than the calculated probability of that outcome. They represent positive expected value opportunities.

Q: How much does it cost to use the APIs?

A: Odds-API and Football-API have free tiers. OpenAI uses a pay-as-you-go model based on tokens used. Check each provider's pricing page for details.

Q: Can I customize the theme colors?

A: The plugin includes two pre-built themes. You can create custom themes by editing CSS files in `/assets/themes/` directory.

Q: Is my API key secure?

A: Yes. API keys are stored in WordPress database and sanitized on input. Never share them. Treat them like passwords.

Q: What happens if I hit my API limit?

A: The plugin stops making calls to that API and shows a friendly message. Data won't display until the limit resets or you increase your quota.

Q: Can I display recommendations on multiple pages?

A: Yes! Use shortcodes on any page/post or add widgets to multiple sidebar areas.

Q: Is there mobile support?

A: Yes, the plugin is fully responsive and works on desktop, tablet, and mobile devices.

Q: Can I export betting data?

A: Yes, all widgets include a "Export to CSV" button for downloading data as a spreadsheet.

Q: What if I encounter a bug?

A: Check the Activity Log for error details. Enable debug mode in Settings. Consult the Troubleshooting section or contact support.
